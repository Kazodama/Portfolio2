Qui suis-je ? Thomas Aretin - 26 ans - Permis B - Dijon

Après 4 ans à travailler dans le commerce, j'ai changé de secteur d'acitivité pour m'orienter vers le développement. J'ai validé une formation en tant que Concepteur, Développeur d'Applications (CDA) au GRETA 21 de Dijon en 2021.
Passionné et curieux de nature, j'ai commencé par développer des jeux 2D sur Unity. Durant la formation, j'ai acquis des connaissances dans le développement d'applications desktop ( C#, Windows Form, Unity 3D), web ( HTML, CSS, PHP, Bootstrap ) et mobile ( Android Studio ). J'ai aussi mis en place des bases de données MySQL au cours de ces différents projets. Après le diplôme obtenu, j'ai continué à me former chez moi sur des frameworks ( Code Igniter, Symfony ) et certains logiciels comme Unity.
Aujourd'hui je cherche un poste de développeur back-end ou Unity.
