A propos de moi

J'ai travaillé 4 ans en tant que chef de secteur en grande distribution. Ensuite, j'ai changé de secteur d'acitivité pour m'orienter vers le développement. J'ai validé une formation en tant que Concepteur, Développeur d'Applications (CDA) au GRETA 21 de Dijon en 2021.
Passionné et curieux de nature, j'avais déjà développé des jeux 2D sur Unity. Durant la formation, j'ai acquis des connaissances dans le développement d'applications desktop ( C#, Windows Form, Unity 3D), web ( HTML, CSS, PHP, Bootstrap ) et mobile ( Android Studio ). J'ai aussi mis en place des bases de données MySQL au cours de ces différents projets. Après le diplôme obtenu, j'ai continué à me former chez moi sur des frameworks ( Code Igniter, Symfony ) et certains logiciels comme Unity.

Aujourd'hui je cherche un poste de développeur fullstack, C# ou Unity.

Unity

Simulateur de dragage en réalité virtuelle

Simulateur réalisé sur Unity durant mon stage dans l'entreprise AC2F. Le simulateur est déstiné aux apprenants de la formation Conduite d'engins hydrauliques dispensée au GRETA de St Jean de Losne.

Une fois le casque de réalité virtuelle sur la tête, on pilote la pelle hydraulique avec des joysticks. Le but est de draguer les algues hors de l'eau, sans endommager l'engin. Les données de chaque session sont enregistrées dans une base de données MySQL, on peut les consulter sur un site web. Les scripts Unity sont en C# et les composants d'accès auxc données en PHP Nous avons développé cela à 2, en intégralité, sur 3 mois.

Ci dessous vous pouvez retrouver une vidéo test.
